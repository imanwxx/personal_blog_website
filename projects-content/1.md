# 个人博客系统

这是一个基于 Next.js 和 Tailwind CSS 构建的现代化个人博客系统。

## 技术架构

### 前端框架

```typescript
// Next.js App Router
export default function BlogPage() {
  return (
    <main>
      <h1>欢迎来到我的博客</h1>
    </main>
  );
}
```

### 样式系统

使用 Tailwind CSS 实现响应式设计和暗色主题。

## 核心功能

### 1. 文章管理

- Markdown 文章支持
- 文章分类和标签系统
- 搜索功能
- 精选文章展示

### 2. 项目展示

- 项目卡片展示
- 项目详情页面（支持 Markdown 内容）
- 技术栈标签
- GitHub 和在线演示链接

### 3. 随笔系统

- 简洁的随笔列表
- 日期排序
- 快速浏览

### 4. 管理后台

- 文章上传
- 项目管理
- 轮播图配置
- 关于我编辑

## 特色功能

### 3D 动画组件

博客集成了多个 3D 动画组件：

- 自动驾驶汽车
- AI 神经网络
- 机器人模型

这些组件使用 Three.js 实现，为用户提供了沉浸式的视觉体验。

### 图片轮播

支持管理员上传的首页轮播图，自动轮播展示精选内容。

## 部署方案

项目支持多种部署方式：

1. **Docker 容器化部署**
2. **静态网站托管**
3. **云服务器部署**
